# E-Waste

## Sobre el proyecto
Nos preocupamos por el manejo de los desechos eléctricos y electrónicos. Creamos una app que te facilita y te comunica con el recolector de estos desecchos. 
Por nuestro ambiente, por el mundo..

¡Recicla!

![about](https://github.com/x-devs/hack/blob/master/assets/e-waste_web-01_720.png)

La aplicación le permite al usuario reportar si existen desechos eléctronicos en una zona de su comunidad, también  solicitar que la empresa o persona encargada la recolección que recoja el desecho hasta la casa o punto dónde se ubique el usuario y además brinda información sobre centro de acopio cercanos a mi ubicación.
